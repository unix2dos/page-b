一 简到崩溃的欧洲史（一）：希腊&罗马

  • 📌 米诺斯在希腊克里特岛上，这就是希腊文明1.0时代：克里特文明。

    • ⏱ 2023-08-05 10:01:14

  • 📌 岛上的米诺斯慢慢不行了，希腊本土开始雄起，进入文明2.0时代：迈锡尼文明。

    • ⏱ 2023-08-05 10:01:24

  • 📌 于是希腊一堆小国家，在两位大哥的带领下，居然齐心协力把波斯给赶走了，这就是欧亚奇迹大碰撞——希波战争

    • ⏱ 2023-08-05 10:02:58

  • 📌 人一闲下来，就容易回忆往事。雅典和斯巴达就闲得发慌，带着小弟继续关上门窝里斗，发生了希腊内战：伯罗奔尼撒战争。

    • ⏱ 2023-08-05 10:03:11

  • 📌 亚历山大玩儿命征服四面八方，希腊还不够他塞牙缝，于是他一路向东征服了埃及、波斯、西亚和印度，建立了地跨欧、亚、非三大洲的超级帝国。

    • ⏱ 2023-08-05 10:05:07

  • 📌 虽然罗马征服了希腊，但很快变成了希腊文化的拥趸，继承下来并发扬光大，造就了欧洲文明。

    • ⏱ 2023-08-05 10:05:59

  • 📌 “哥你说的是——罗马王政，罗马共和国，还是罗马帝国啊？” 一句话打败全桌93%的朋友。

    • ⏱ 2023-08-05 10:06:39

  • 📌 然后大家觉得，王政真的不是个好制度，于是改由“执政官+元老院”这个男子组合共同执政，这样一来谁也不能为所欲为。

    • ⏱ 2023-08-05 10:06:56

  • 📌 这就是罗马共和国。紧接着，罗马从可怜的小土坡，扩张到了欧、非、亚三洲。从地中海任何一点往左看，是罗马；往右看，仍然是罗马。英国、法国那些地方呢？除了蛮族，啥玩意儿都没有，穷得掉眼泪，最后全让罗马包了圆。

    • ⏱ 2023-08-05 10:07:23

  • 📌 恺撒是死了，但他的外甥女有个儿子叫屋大维，把元老院干掉一半，然后接着玩终身独裁。现在罗马地盘这么大，民族这么多，终身独裁，不就是皇帝吗？ 从此罗马共和国变成了罗马帝国。

    • ⏱ 2023-08-05 10:08:06

  • 📌 有个皇帝叫戴克里先，把帝国分成东、西两块，分别由一对大皇帝和小皇帝管，总共四个皇帝。

    • ⏱ 2023-08-05 10:09:14

  • 📌 加上新来蹭热闹的，最多的时候有六个人自称大皇帝，最后被其中一个全收拾完，罗马重新恢复一个皇帝制。￼这个皇帝就是——君士坦丁大帝。

    • ⏱ 2023-08-05 10:09:30

二 简到崩溃的欧洲史（二）：黑暗的中世纪

  • 📌 所谓中世纪，指的就是西罗马灭亡到文艺复兴这一千年。

    • ⏱ 2023-08-05 10:10:46

  • 📌 日耳曼是个大民族，有很多分支：哥特人、法兰克人、汪达尔人、盎格鲁人、萨克逊人等等。罗马分裂之后，东罗马一直独善其身。日耳曼人专挑西罗马下手。西罗马从此灭亡，取代它的，是无数个日耳曼民族建立的无数个小国家。

    • ⏱ 2023-08-05 10:11:13

  • 📌 这些当年被罗马人称为蛮族的日耳曼人，成了现在大部分欧洲人的祖先。

    • ⏱ 2023-08-05 10:11:26

  • 📌 日耳曼民族分支无数，有一支法兰克人，建立了一个国家叫法兰克王国。它本来只是一堆日耳曼国家中的一个，直到出现了一个超级大佬：

    • ⏱ 2023-08-05 10:11:42

  • 📌 查理曼大帝 （约公元800年，大致是中国唐朝时期）他把几乎所有的日耳曼国家都揽到一起，自己也成了整个西欧的皇帝。于是继西罗马之后，西欧又统一了起来。

    • ⏱ 2023-08-05 10:12:04

  • 📌 扑克牌里的这张红桃K，就是查理曼大帝。

    • ⏱ 2023-08-05 10:12:19

  • 📌 它们最后变成了法国、意大利、德国。

    • ⏱ 2023-08-05 10:12:46

  • 📌 欧洲一分为三，东法兰克慢慢变成了德国的雏形：德意志。它跟别的国家不一样，更像我们的春秋战国，里面全是诸侯，各自为政。其中有一个诸侯混大了，不但在德意志当了大哥大，还冲出来欺负别的小伙伴，比如没事就窜到隔壁砸意大利的场子。￼这就是奥托大帝。 （约公元960年，大致是中国北宋初年）

    • ⏱ 2023-08-05 10:13:53

  • 📌 这位奥托大帝觉得自己贼厉害，就跟当年罗马帝国一样厉害，干脆把德意志改叫罗马帝国。他的继任者还嫌不够雄壮，又加了“神圣”二字。

    • ⏱ 2023-08-05 10:14:04

  • 📌 就这样，传说中的神圣罗马帝国诞生了。这是德意志民族在历史上的第一次人生巅峰，所以又叫第一帝国。

    • ⏱ 2023-08-05 10:14:09

  • 📌 早在查理曼他爷爷的时候（大致是我们唐朝时期），中东就有一群人，跟欧洲人因为信仰不一样，闹了点小矛盾。后来他们还抢了一块很重要的地段：耶路撒冷。吵个嘴就算了，抢地盘这种事不能忍。西欧的兄弟们纷纷抄家伙，组团冲去跟中东人拼命，双方做梦都想打败对方

    • ⏱ 2023-08-05 10:16:26

  • 📌 这就是十字军东征，不过闹了两百年没啥收成，最后全被赶回了欧洲。（1096年—1291年，大致是中国宋朝时期）

    • ⏱ 2023-08-05 10:16:34

三 简到崩溃的欧洲史（三）：下海的欧洲

  • 📌 英国成了最后的欧洲一霸，抢到几乎所有亚洲和北美的殖民地。￼从此它的地盘遍布全世界，24小时都能见到太阳，号称日不落帝国。

    • ⏱ 2023-08-05 10:20:41

  • 📌 国王终于犯了众怒，老百姓一言不合，抓住他送上了断头台。这就是著名的——法国大革命。没有国王的法国乱成一片，一个人间小钢炮趁乱上位。

    • ⏱ 2023-08-05 10:22:18

  • 📌 拿破仑是个有着巨大野心的军事天才，刚当上皇帝就四处开炮，几乎整个欧洲都变成了他们家，这就是拿破仑时代。他只在一个地方吃了亏：俄国。

    • ⏱ 2023-08-05 10:22:34

  • 📌 最后普鲁士憋了个大招：铁血宰相俾斯麦。外号听上去贼硬气，其实他前世是个厨子。此男最喜煽风点火添油加醋，他利用这个政治特长打败自家兄弟奥地利，打败隔壁法国，统一德国，变成欧洲第一强国，号称第二帝国

    • ⏱ 2023-08-05 10:24:05

  • 📌 于是形成了德国&奥匈帝国&意大利的“同盟国”。英国&法国&俄国一看，哎哟学会合体了，我们也合，这就合出来一个“协约国”。

    • ⏱ 2023-08-05 10:24:45

  • 📌 结果一打仗，意大利就叛变到对方阵营里了。

    • ⏱ 2023-08-05 10:25:57

    • ⏱ 2023-08-05 11:40:00

八 美国往事（一）：独立战争

  • 📌 这个时期，法国在欧洲已经是大哥了，来美洲纯属玩票，而英国人不一样，真心指着美洲捞钱。混子曰：真爱战胜一切，所以最终英国战胜了法国，占领了几乎整个北美。版图大致是现在的加拿大+美国。

    • ⏱ 2023-08-05 11:45:25

  • 📌 你看，就为了多卖点儿茶叶，英国人损失了一整个美国，多出来的茶叶为啥非要卖掉？这个故事告诉我们：别捡了芝麻丢了西瓜。为了祝贺美国人独立，法国还幸灾乐祸地送了个大礼。￼纽约的自由女神像，就是法国人在美国独立100周年的时候送的，象征自由和反抗暴政。所以交朋友一定要交法国这样的，还送你女神。

    • ⏱ 2023-08-05 11:47:43

十 美国往事（三）：上帝保佑美国

  • 📌 美国人在战争中谁也不想得罪，保持中立，然后毫无节操地两边卖武器火上浇油，不但让欧洲自己消耗个精光，还成功地薅走了整个欧洲的钱，忽然就成了世界级土豪。
