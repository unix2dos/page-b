前言 娜拉出走之后

  • 📌 正如大多数爱情电影都终结于男女主角步入“婚姻的殿堂”，很长
    一段时间里，对政治发展的分析也大多终结于“民主转型”的时刻。许多人似乎相信，“民主转型”意味着政治发展划上一个完美的句号，正 如人们假定男女主角手牵手以后会是“从此过上了幸福的生活”。
    这种假定的一个后果，就是很长一段时间里，“政治转型”研究主
    要都是政治转型的“动力学”研究。

    • ⏱ 2024-06-15 09:04:16

  • 📌 步入婚姻的殿堂”之后未必是“幸福的生活”，有时候甚至“婚姻是爱情的坟墓”。民主转型之后出现的也未必是民主稳固，有可能是民主倒退和崩溃，甚至可能是战火纷飞。“句号”并非真的句号，而是意味深长的省略号，对这个省略号，我们所知 甚少。

    • ⏱ 2024-06-15 09:04:28

  • 📌 《经济学人》在2014年的一篇社论中写道:“民主正在经历一个艰难时世。在那些独裁者被赶跑的地方，反对派大多没能建立一个行之有效的民主政府。即使在那些民主得以建立的地方，系统的缺陷令人担忧地显现，对政治的幻灭也随处可见。

    • ⏱ 2024-06-15 09:04:52

  • 📌 “阿拉伯之春”中的民主转型遭遇了几乎是全军覆没式的失败。在拉美，民主转型与其民粹主义传统的结合致使数个大国出现经济危机甚至崩溃。在非洲多个国家，周期性民主选举成为周期性族群冲突的导火索，以至于在某些地方投票成为骚乱集结 号。甚至在西方的边陲地带一中东欧、南欧、西亚，似乎已经尘埃 落定的民主转型被重新打上问号:东欧的民粹主义崛起、南欧的债务危机、土耳其的民主倒退都为“从此过上了幸福的生活”之叙事蒙上了巨大阴影。

    • ⏱ 2024-06-15 09:05:08

  • 📌 反讽的是，从历史上被倾覆的民主来看，或从当代新兴民主的命运
    来看，在漫长艰难的转型过程中，转型最致命的陷阱未必是“权欲熏心的独裁者”，而常常是“完美民主”的诱惑:人们以“真民主”的名义去摧毁所谓“假民主”，用“更深民主”的名义去否定所谓“表像民主”，用“大众民主”的名义去攻击所谓“精英民主”。总之，以民粹式民主去摧毁自由式民主。换言之，自法国大革命以来，摧毁民主的就常常不是专制，而恰恰是民主自身失去节制。

第一章 第三波民主化的进展与受挫

  • 📌 墨西哥的民主转型一开始比较平稳。然而，失控的暴力组织，尤其是毒品黑帮，成为墨西哥社会的顽疾。为解决这个问题，墨西哥总统卡尔德龙(FelipeCalderon)从2007年开始发动了“缉毒战争”。然而缉毒战争不但没能消灭黑帮，反而演变成了一场近似于多方内战的僵局。一方面，警察力量的军队化成为一种趋势，过度抓捕、过度使用暴力、血腥打压成为常态;另一方面，为报复政府，黑帮展开越来越多的无差别袭击，其行为的恐怖主义色彩越来越显著，结果是成千上万人在夹击中丧 生。正是因为这种暴力漩，墨西哥的政治自由和公民权利也受到侵蚀。

    • ⏱ 2024-06-15 09:09:24

  • 📌 21世纪的俄罗斯经典地体现了“中间政体陷阱”。在今天的俄罗
    斯，选举真实存在，甚至有一定的竞争性。2018年的总统选举，普京赢得了76.7%的选票，也就是说，不但有其他政党的候选人参选，而且这些候选人加起来还赢得了20%多的选票(其中共产党候选人赢得了11.8%的选票，构成最大反对派)。但是，另一方面，俄罗斯政治自由受到严重限制。

    • ⏱ 2024-06-15 10:36:00

  • 📌 泰国是“政体震荡”的典型。自20世纪70年代中期以来，泰国民主
    已经有过“四上四下”。1947-1973年，泰国一直是军人当政。1973年由学生领导的“10月14日抗议”导致了军人政权垮台。然而，这次革命带来的政治成果是极其短暂的一20世纪70年代中期，越南、柬埔寨、老 挝等国红色政权的胜利引发了泰国的反共热潮，1976年军队再次掌权。

    • ⏱ 2024-06-15 21:51:19

  • 📌 哪怕今天，通过政变上台的巴育(PrayutChan-o-cha)总理权力也并不稳固。理论上，他是个“威权统治者”，但是其“威权统治”同样岌岌可危。2019年的大选中，他利用“在任优势”和军方力量影响选举结果，但只赢得了大众选票的23.7%，比第二赢家只多了1.5%。威权操控下也只有如此微弱的优势，可以想见，泰国的政治撕裂还在继 续，新一波的政治动荡可能再次到来。

    • ⏱ 2024-06-15 21:52:24

  • 📌 以韩国为例，1987年最近一次民主转型之前，韩国政治过山车一
    般大起大落。从1948年确立第一共和国，到1960-1961年第二共和国时期短暂而动荡的民主实验，到朴正熙时期越来越稳固的独裁统治(第三、第四共和国)，再到全斗焕和卢泰愚时期逐渐失控的局势(第五共和国)，韩国战后的四十多年，无论是民主体制还是威权体制，似乎都难以持久地维系政治稳定。然而，从1987年的民主转型开始，至今三十余年，韩国“反常”地维持了相对的稳定和发展，尽管领导人频繁受 审，民主政体本身却未遭遇系统性挑战。

    • ⏱ 2024-06-15 21:53:46
